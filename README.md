# 🌐 Cobalt Project Application

An interactive application built to analyze and visualize **USGS Cobalt project data**. This project integrates **SQL, database management, and data visualization** techniques to deliver meaningful insights about cobalt resources, locations, and production.

---

## 📖 Overview
The Cobalt Project Application provides:
- A **Java Swing-based desktop UI** for working with datasets.
- Integration with **MySQL database** for querying and analyzing cobalt data.
- Support for both **original** and **normalized** tables.
- Execution of stored procedures for **data exploration and reporting**.
- Visualizations and demos for presenting project outcomes.

---

## 🚀 Features
- 📊 **Data Visualization** – Display and compare normalized vs. original datasets.  
- 🗄️ **Database Connectivity** – Connects to a MySQL backend using embedded SQL.  
- ⚡ **Stored Procedures** – Run pre-defined queries to generate meaningful reports.  
- 🎥 **Video Demo** – Walkthrough included in the `Video Demo/` folder.  
- 📂 **ERD & Normalized Data** – Entity Relationship Diagram and normalization steps included.
